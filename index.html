O que é uma tags?

As tags do HTML são usadas para informar ao navegador a estrutura da página. Com elas o navegador identifica se o texto é um título, um parágrafo, um link, uma imagem, entre outros. Essa tags, normalmemte, tem uma abertura e um fechamento .

Tags HTML de título

Títulos são definidos com as tags h1 a h6. A h1 define o título maior. A h6 define o título menor. A HTML automaticamente adiciona uma linha em branco extra antes e após um titulo.

Tags HTML de textop

Os parágrafos são definidos com a tag p. A HTML adiciona automaticamente uma linha em branco extra antes e após um parágrafo.

Tags HTML de linka

As tags dos links HTML (que criam o que é conhecido como “hiperlink”) são usadas comumente em links externos (para outra página ou documento em um site diferente) ou internos (para outros recursos do site), é a tag a.

Tag HTML de imagemimg

A tag HTML img faz parte do conjunto de elementos da linguagem de marcação utilizados para exibir conteúdos visuais em uma página. Entretanto, sua utilização vai além de apenas embelezar o site, pois as imagens podem ter outras funções, como ser aplicadas em links ou para ilustrar botões.
