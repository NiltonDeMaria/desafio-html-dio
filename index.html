O que é uma tags?

As tags do HTML são usadas para informar ao navegador a estrutura da página. Com elas o navegador identifica se o texto é um título, um parágrafo, um link, uma imagem, entre outros. Essa tags, normalmemte, tem uma abertura e um fechamento .
